My Softwares

In my leisure, I work on several open source project. Those mostly to ease my work, but not directly related to my job. Here are few of them.

mkBiB

mkBiB is a bibtex file creator for based on GTK+-3. User can simply create a bibtex entry filling up the input entries or he can make a "google scholar" search within the same interface, providing Author and/or Title(as key word) and Year as optional field.

You can download the stable version from CTAN. The latest version will always be available from Savannah(non-gnu). This package is under GPLv3 license. Users are free to use the program.

I gratefully acknowledge the guidance and help I received from untiring members of gtkforums.

ptbl

ptbl is a Periodic Table package for linux system based on GTK+-2. Each element shows a list of physical/chemical properties, mainly focusing at the need of Condensed Matter Physicist and Material Scientist.

You can download the program from Savannah(non-gnu). This package is under GPLv3 license. Users are free to use the program.

Properties of elements are taken form http://www.periodictable.com

Shadow: Gnome Icon theme

This is a gnome icon theme; within 6 months of its release, its among the top 30 highest rated icon theme at gnome-look.org

This work is available for free. Download it from gnome-look or directly from github.

vimf90: a fortran IDE for vim

This is a fortran90+ IDE for vim.
